# BMI Calculator

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
It allows users to calculate their BMI quickly and understand their health category (Underweight, Normal, Overweight, or Obese) based on their height and weight.

---

## Features

- Instant BMI calculation  
- Clean and responsive UI  
- Displays BMI category and result interpretation  
- Beginner-friendly JavaScript logic  
- Lightweight and fast (no external dependencies)
